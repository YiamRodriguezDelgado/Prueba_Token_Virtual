## Quick start

- Clone the repo: `https://github.com/YiamRodriguezDelgado/Prueba_Token_Virtual.git`. or download ZIP.
- In the root of the project, run `npm install`.

## Development Server

- In the root of the project, use `npm start` to run the project.
- By default, the application will run on <b>http://localhost:3000/`</b>.
